== Description ==

Este não é só um plugin, simboliza a esperança e o entusiasmo de uma geração inteira resumidos em frases do fundador do Movimento Escoteiro Lord Robert Baden Powell.
Sempre Alerta para Servir e Viver Melhor Possível!. Quando ativado, você verá aleatoriamente frases de Baden-Powell, BP no canto superior direito da tela de administração em cada página.
Obrigado aos voluntários pela arte.
